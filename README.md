## Dashboard de Saúde Mental e Burnout entre Estudantes
## Funcionalidades
- O dashboard foi feito com os dados retirados do Kaggle e com o objetivo de visualizar gráficos analisando os níveis de burnout por gênero,curso,analisando qualidade de sono vs burnout e nível de estresse vc burnout.
## Como executar o código
- Como rodar o código: Instale a extensão python no VSCode,depois instale no terminal o streamlit(pip install streamlit plotly),rode o código com os comandos 'python -m streamlit run main.py'.
# Dashboard Interativo
- O dashboard possui filtros interatidos para analisar só: Gênero,Curso e Níveis de Burnout.
